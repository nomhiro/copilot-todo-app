# Copilot TODO App

GitHub Copilot認定資格コースのハンズオン用TODOアプリケーション

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Features

- TODO CRUD operations
- Priority filtering (High/Medium/Low)
- Search functionality
- Statistics dashboard

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS

## Project Structure

```
src/
├── app/              # Next.js App Router pages
├── components/       # React components
├── lib/
│   ├── types/        # TypeScript type definitions
│   ├── utils/        # Utility functions
│   ├── validators/   # Input validators
│   └── data/         # Mock data
└── __tests__/        # Test files
```

## Hands-on Exercises

This app is designed for GitHub Copilot certification course hands-on exercises.

### Available Exercises:
- Section 5: Copilot Chat commands (/explain, /tests, /fix)
- Section 6: Coding Agent tasks
- Section 7: PR Summary and Code Review
- Section 9: CLI commands
- Section 11: Prompt Engineering
- Section 12: Debugging and Documentation
- Section 13: Test Generation
- Section 14: Content Exclusion
